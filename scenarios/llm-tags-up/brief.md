# LLM Auto-Tagging Pitch

We want to deploy an LLM to auto-generate search-term tags for our product catalog. Right now the tags are hand-written by our ops team, and we think an LLM would do a more comprehensive and more consistent job. Tags drive site-search recall, so better tags should mean more relevant results and better conversion. We'd also free up some ops capacity since they spend a lot of cycles on this. I'd like approval to move from the pilot we ran into a phased production rollout.
